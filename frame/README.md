# frame

Frame helps refine system design, product architecture, runtime boundaries, config models, deployment models, operational models, and implementation plans through one focused question at a time.

## Known Limits

- Does not produce full implementation plans unless explicitly asked for a summary or draft.
- Depends on the user or repo context for domain facts; it should not invent missing constraints.
- Challenges weak assumptions, but final product and architecture decisions remain the user's responsibility.

## Install

```bash
npx skills add https://github.com/yarlson/skills/tree/main/frame
```
